> founder orchestrating Claude — so we are both a Claude customer and a proof point for
> what a solo founder can ship with Claude.

**How will you use the credits?**
> Credits go directly into the product's core loop. Pre-revenue, the binding constraint

> customers, (2) increase planning quality with longer context and adaptive thinking,
> and (3) add a verification pass on outbound content before it's sent. This is not
> infra overhead — it is the product.
