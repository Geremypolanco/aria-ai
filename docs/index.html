FAQ

Questions answered

How is ARIA different from an AI assistant like ChatGPT? +
ChatGPT waits for you to ask it questions. ARIA runs autonomously 24/7 without you doing anything. It actively searches for clients, creates products, publishes content, and executes revenue strategies — you just watch the results.
Do I need technical skills to use ARIA? +
No. You connect your accounts (Stripe, GitHub, email) once during setup — ARIA handles everything else. If you want more control, the API is available on the Agency plan.
What accounts does ARIA need to work? +
At minimum: a GitHub account (free) and a Telegram account (free) to receive alerts. For full power: Stripe (free), Gumroad (free), and an email/SMTP account. ARIA can even self-onboard onto platforms using your credentials.
Can ARIA actually send emails and find real clients? +
Yes. With SMTP credentials configured, ARIA sends real emails to real businesses it finds via web search. It researches each company's specific pain points and writes personalized proposals. This is not mass spam — it's targeted, research-backed outreach.
What if I don't like the strategies ARIA runs? +
ARIA's Thompson Sampling algorithm learns which strategies work for YOUR business and automatically runs them more. You can also view and influence strategy weights via the dashboard.
Is there a free trial? +
Yes — 7 days free on any plan, no credit card required to start. Cancel before day 7 and you pay nothing.
