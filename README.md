# 🧠 ARIA AI — El Sistema Operativo de Negocio Autónomo (Aria OS)

Aria ha evolucionado de un sistema de agentes a una **Organización Autónoma Digital (DAO)** completa. Esta arquitectura final consolida todas las capacidades operativas, estratégicas y económicas en un núcleo unificado: `aria_os`.

## 🧬 Estructura del Sistema (Aria OS)

Aria se divide en 4 kernels principales y 3 capas de soporte que permiten una autonomía total:

### 1. Cognition Kernel (El Cerebro)
- **Opportunity Scoring**: Evalúa mercados y competidores con modelos bayesianos (**PyMC**).
- **Strategy Optimization**: Refina sus propios prompts y tácticas mediante **DSPy**.
- **GraphRAG**: Razonamiento profundo sobre el conocimiento organizacional.

### 2. Economic Kernel (El Corazón)
- **Revenue Autonomy**: Gestión directa de **Stripe** y **Shopify**.
- **Economic Attribution**: Seguimiento de ROI por agente y canal usando **DuckDB** y **PostHog**.
- **Financial Decisioning**: Reasignación automática de presupuestos según rendimiento.

### 3. Execution Kernel (Los Músculos)
- **Swarm Economy**: Orquestación de enjambres de agentes (**CrewAI**, **AutoGen**) coordinados por **Temporal**.
- **Experimentation Engine**: Ciclos autónomos de Idea → Test → Métrica mediante **GrowthBook**.

### 4. Perception Layer (Los Sentidos)
- **World Observation**: Monitoreo proactivo de tendencias globales (**GDELT**, **OpenAlex**).
- **Autonomous Browsing**: Operación real de internet con **Browser Use** y **Firecrawl**.

---

### 🏛️ Gobernanza y Memoria Evolutiva
- **Memory Core v2**: Integración de **Neo4j**, **Graphiti** y **Mem0** para una memoria histórica económica.
- **Self-Improvement Loop**: Evolución controlada del código fuente mediante **Aider** y **SWE-agent**.
